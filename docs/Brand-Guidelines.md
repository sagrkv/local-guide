# Boko - Brand Guidelines

## Logo

### Primary Logo
The Boko logo consists of the word "SUMMER" wrapped in code brackets `{BOKO}`.

### Logo Variations
| File | Usage |
|------|-------|
| `boko-logo-dark.svg` | Use on dark backgrounds |
| `boko-logo-light.svg` | Use on light backgrounds |
| `boko-full-dark.svg` | Full logo with "APP" tagline (dark bg) |
| `boko-full-light.svg` | Full logo with "APP" tagline (light bg) |
| `boko-favicon.svg` | Favicon, app icon: `{B}` |

### Clear Space
Maintain minimum clear space around the logo equal to the height of the brackets.

### Minimum Size
- Digital: 120px width minimum
- Print: 30mm width minimum

### Don'ts
- Don't stretch or distort the logo
- Don't change the colors outside brand palette
- Don't add effects (shadows, gradients, outlines)
- Don't place on busy backgrounds without contrast

## Color Palette

### Primary Colors

| Color | Hex | RGB | Usage |
|-------|-----|-----|-------|
| Charcoal | `#0A0A0A` | 10, 10, 10 | Primary background |
| White | `#FFFFFF` | 255, 255, 255 | Primary text |
| Orange (Dark) | `#FF9500` | 255, 149, 0 | Accent, CTAs, brackets |
| Orange (Light) | `#E07800` | 224, 120, 0 | Accent on light backgrounds |

### Secondary Colors

| Color | Hex | RGB | Usage |
|-------|-----|-----|-------|
| Dark Gray | `#1A1A1A` | 26, 26, 26 | Cards, surfaces |
| Medium Gray | `#666666` | 102, 102, 102 | Secondary text |
| Light Gray | `#888888` | 136, 136, 136 | Muted text, taglines |

### Color Usage Rules
- Primary background is always `#0A0A0A`
- Orange is the primary accent - use sparingly for impact
- Maintain 4.5:1 contrast ratio minimum for accessibility

## Typography

### Primary Typeface
**Space Grotesk** - Google Fonts

A modern geometric sans-serif that balances technical precision with friendly approachability.

### Font Weights
| Weight | Name | Usage |
|--------|------|-------|
| 400 | Regular | Body text, brackets |
| 500 | Medium | Subheadings, taglines |
| 600 | SemiBold | Logo text, headings |
| 700 | Bold | Hero headlines |

### Type Scale
```
Hero:     72px
H1:       48px
H2:       36px
H3:       24px
Body:     16px
Small:    14px
Caption:  12px
```

### Line Heights
- Headings: 1.2
- Body: 1.5
- Tight: 1.1

### Letter Spacing
- Logo text: 2px
- Tagline "APP": 8px
- Body text: 0

## Voice & Tone

### Brand Personality
- **Professional** but not corporate
- **Technical** but accessible
- **Confident** but not arrogant
- **Direct** but friendly

### Writing Guidelines

**Do:**
- Use clear, simple language
- Be direct and to the point
- Show technical competence naturally
- Use active voice
- Be honest about capabilities

**Don't:**
- Use jargon without explanation
- Over-promise or use hyperbole
- Sound robotic or overly formal
- Be vague about pricing or process

### Example Phrases
✓ "We build websites that convert"
✓ "No templates. No bloat. Just clean code."
✓ "Tell us what you need. We'll give you a straight answer."

✗ "We leverage cutting-edge synergies..."
✗ "Our best-in-class solutions..."
✗ "Revolutionary AI-powered..."

## Imagery

### Photography Style
- High contrast
- Minimal, clean compositions
- Tech-forward aesthetic
- Real people, authentic moments

### Illustrations
- Geometric, minimal style
- Line art preferred
- Orange accent color for highlights

### Icons
- Line icons (not filled)
- 2px stroke weight
- Rounded corners
- Consistent sizing (24px standard)

## Applications

### Website
- Dark theme primary
- Orange accents for CTAs and highlights
- Space Grotesk throughout
- Smooth, purposeful animations

### Social Media
- Dark backgrounds
- Logo or favicon as profile picture
- Consistent orange accent usage

### Email Signatures
```
Name
Role, Boko
hello@boko.app
```

### Business Cards
- Dark background (`#0A0A0A`)
- White text
- Orange accent line or logo brackets
- Minimal information
